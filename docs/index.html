Code

These are links to some of the IPFS repositories.

  • github.com/ipfs/ipfs - original IPFS paper. Soon the spec too.
  • github.com/ipfs/go-ipfs - the main, reference implementation
  • github.com/protocol/ipfs-webui - our web console
  • Search github for "ipfs"

Community

The IPFS community maintains the following communication channels:

  • github: github.com/ipfs/ipfs for the IPFS spec
  • github: github.com/ipfs/go-ipfs for the go-ipfs implementation
  • IRC: #ipfs on chat.freenode.net for live help + some dev discussions (Logs)
  • Google Group: ipfs-users@groups.google.com
  • Twitter: @IPFSbot for some news
